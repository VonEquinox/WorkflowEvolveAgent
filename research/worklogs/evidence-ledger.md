## Audit Notes

1. **原计划的文献编号大多正确**。截至 2026-07-13，`2605.27621` 已经真实存在，并于 2026-05-26 提交，不再应标为“编号存疑”。
2. **最大变化是 novelty landscape**：EvoMAC（ICLR 2025）、SEW（2025）、FlowReasoner/MaAS（2025）、MermaidFlow（ICLR 2026 submission）、DGM（2025）、Socratic-SWE（2026）使“无人做 coding workflow 自进化 / 图 IR / trace→skill”这类宽泛表述不成立。
3. **仍可能成立的窄创新点**：在 repository-level coding agent 上，把可观测 execution graph、dependency-sound content-addressed result reuse、single-trace attribution proxy（经 LOO 校准）、typed executable workflow IR、retrieve-and-constrained-edit、champion gate 串成一个统一系统，并对 token/latency/pass-rate 做端到端消融。
